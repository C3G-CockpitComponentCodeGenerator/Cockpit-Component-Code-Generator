## [1.0.0] - 2026-07-16

### Initial Public Release

#### Features

- Automatic Firmware Generation
- Automatic Pin Allocation
- Manual Pin Allocation
- Interactive GPIO Pin Map
- MCP23017 Expansion Support
- Board Health Dashboard
- Firmware Preview
- Wiring Documentation
- MAX7219 Seven Segment Support
- I²C Display Support

#### Supported Components

- Push Buttons
- Toggle Switches
- Rotary Encoders
- Rotary Switches
- Analog Inputs
- LEDs
- MAX7219 Displays

#### Validation

- Hardware Tested
- Production Ready
